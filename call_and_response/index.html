How to use

This web-based ear trainer offers two output options: a built-in synthesizer or a connected MIDI device. The default output is the synthesizer.

Configure the ear trainer to suit your needs by modifying the YAML file under "Settings". As exercises are played, the notes for each measure can be displayed, providing a visual reference to reinforce auditory recognition.

To switch to a MIDI device, find its ID in the "MIDI Devices" list. Replace "Synth" under "MIDI device index" in the settings with this ID number.
